<task type="auto">
  <name>Task 2: Create promo image generation script and generate 440x280 PNG</name>
  <files>scripts/promo-image.mjs, docs/store/promo-440x280.png</files>
  <action>
1. Create `docs/store/` directory if it doesn't exist.

2. Create `scripts/promo-image.mjs` — an ES module script that:
   a. Reads the extension icon from `extension/icons/icon-128.png` and converts to base64 for SVG embed (avoids emoji rendering issues with libvips — per research pitfall 2)
   b. Constructs an SVG string at exactly 440x280 with:
      - Dark gradient background matching extension dark mode: `#111827` to `#1f2937` (diagonal)
      - Centered extension icon as base64 `<image>` element (NOT emoji — libvips has no emoji font)
      - Product name "Chat Signal" in white/near-white (`#f9fafb`), ~22px, font-weight 600
      - Tagline "See what your chat is saying" in muted gray (`#9ca3af`), ~14px
      - Font stack: `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
   c. Uses sharp to convert SVG buffer to PNG at exactly 440x280:
      ```javascript
      await sharp(Buffer.from(svg))
        .resize(440, 280, { fit: 'fill' })
        .png()
        .toFile(path.resolve(__dirname, '../docs/store/promo-440x280.png'));
      ```
   d. Logs success message with output path.

3. Run the script: `node scripts/promo-image.mjs`

4. Verify output dimensions using sharp itself or `file` command:
   ```bash
   node -e "import('sharp').then(s => s.default('docs/store/promo-440x280.png').metadata().then(m => console.log(m.width, m.height)))"
   ```
   Must output `440 280`.

Key design decisions (per user decisions + research):
- No platform logos (YouTube/Twitch) — avoids trademark issues entirely
- Dark gradient matches extension dark mode aesthetic
- Keep text minimal — CWS guideline says "avoid text" for promo images
- Use base64 icon embed, NOT emoji text in SVG
  </action>
  <verify>
- `ls -la docs/store/promo-440x280.png` — file exists
- `node -e "import('sharp').then(s => s.default('docs/store/promo-440x280.png').metadata().then(m => console.log(m.width + 'x' + m.height)))"` outputs `440x280`
- `node scripts/promo-image.mjs` runs without errors (idempotent regeneration)
  </verify>
  <done>Promotional image exists at docs/store/promo-440x280.png at exactly 440x280 px with dark gradient, Chat Signal branding, extension icon, and tagline. Script is reproducible for future regeneration.</done>
