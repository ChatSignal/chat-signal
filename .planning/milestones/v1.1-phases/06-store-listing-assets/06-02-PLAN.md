  <action>
Create `scripts/screenshot.mjs` — an ES module Playwright script that generates 3 screenshots. The script must handle these concerns:

**1. Chrome API stubs (CRITICAL — prevents sidebar.js crash):**
Before navigating, inject chrome API stubs via `page.addInitScript()`:
```javascript
await page.addInitScript(() => {
  window.chrome = {
    runtime: {
      sendMessage: () => {},
      onMessage: { addListener: () => {} },
      getURL: (p) => p,
    },
    storage: {
      local: {
        get: (_keys, cb) => { if (cb) cb({}); return Promise.resolve({}); },
        set: (_items, cb) => { if (cb) cb(); return Promise.resolve(); },
      },
    },
    sidePanel: {},
  };
});
```

**2. Viewport and dark mode:**
```javascript
await page.setViewportSize({ width: 1280, height: 800 });
await page.emulateMedia({ colorScheme: 'dark' });
```

**3. Sidebar width presentation:**
The sidebar is designed as a narrow panel (~420px). At 1280px viewport it would look stretched. The screenshot script must set body styling to present the sidebar at a natural panel width centered on a dark background:
```javascript
await page.evaluate(() => {
  document.body.style.maxWidth = '420px';
  document.body.style.margin = '0 auto';
  document.body.style.background = '#1f2937';
  // Set html background too so sides are filled with dark color
  document.documentElement.style.background = '#1f2937';
});
```
